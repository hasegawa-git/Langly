# Langly 
外国語学習をより楽しく、スマートに。
Azure AI Speechを活用した次世代の語学学習Webアプリケーション。

## 技術スタック
- **Frontend**: Next.js (App Router)
- **Database**: PostgreSQL (Raw SQL)
- **Auth**: NextAuth.js
- **Storage**: Supabase Storage
- **AI**: Azure AI Speech (TTS)
