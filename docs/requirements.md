# 要件定義書

## 概要
- **サービス名**: Langly
- **コンセプト**: 自分だけのシンプルなリスニング特化型アプリ

## ターゲットユーザー
  自分の学習レベルに合わせた独自の学習リストでリスニング力を向上させたい外国語学習者

## 機能要件
### ユーザー管理機能
- **サインアップ**: メールアドレスとパスワードで新規登録
- **サインイン**: メールアドレスとパスワードで認証
- **サインアウト**: セッションを破棄
- **プロフィール設定**: ユーザー名を設定
### 学習アイテム生成機能
- **テキスト入力**: 音声化したいテキストを入力
- **音声化**: 入力されたテキストの音声を生成
### 学習アイテム管理機能
- **ライブラリ閲覧**: 事前に用意された学習アイテムを閲覧
- **学習アイテム保存**: 生成した学習アイテムを保存
- **学習アイテム一覧表示**: 生成した学習アイテムをリスト形式で確認
- **学習アイテム編集**: テキストを修正
- **学習アイテム削除**: 不要になった学習アイテムを削除
- **マイリスト追加**: お気に入りの学習アイテムをマイリスト化
### 再生機能
- **再生コントロール**: マイリストに登録された音声を再生または一時停止
- **バックグラウンド再生**: iPhoneを閉じても音声が流れる状態の維持(iOS)
- **再生設定**: 
  - 再生速度の調整（0.8x, 1.0x, 1.2xなど）
  - リピート再生、シャッフル再生

## 非機能要件
### 性能
- **スリープ対応**: APIサーバーがスリープしている場合、アプリ側でインジケーターを表示
### セキュリティ
- **SQLインジェクション対策**: すべてのクエリでプレースホルダを使用
- **APIキーの秘匿**: 秘密鍵は絶対フロントエンドに持たせず、バックエンドの環境変数のみで管理
- **認証の安全性**: 他人の学習リストにはアクセス不可
